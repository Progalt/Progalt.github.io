3D Rendering Engine

A valuable experience in not only 3D techniques but also optimisation and more

I had the goal of producing a testbed for some modern 3D rendering techinques and also learn more about the Vulkan API and how it interacts with the GPU.

I took a barebones window created using SDL2, to an advanced 3D scene.

Notable Features:

  • Deferred Rendering
  • Physically Based Material Rendering
  • Cascaded Shadow Maps
  • Diffuse Irradiance using Scene
  • Post Process Stack:
    • FXAA
    • Fog
    • Temporal Anti Aliasing
    • Chromatic Aberration
  • Bloom - Using a downsampling approach for physically accurate results
  • SSAO implemented through the FidelityFX CACAO
  • CPU Frustum Culling
  • A Scene and Component System
  • Custom binary model format
  • Physically based sky (no clouds though)
